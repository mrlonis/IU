INFO-I 101: Week 2 - Lab 1

Name and define the programming terms we learned today.

  • Function - A function is a type of procedure or routine that carries out a task
  • Argument - An argument is a value that is passed to a function
  • String - A string is a sequence of characters
  • Variable - A variable is a container, given a name, that represents a value
  • Integer - An integer is a whole number ranging from negative infinity to positive infinity

Why is it important to keep track of data types?

It is important to keep track of your data types because as a program gets larger and larger, there will be more and more variables to keep track of. It is important because different data types can clash and trigger errors in a program or not work as expected. For example if you defined a variable word1 as "5" and a variable word2 and "10" and tried to run print(word1+word2) the result would be "510" instead of "15".

What does the len() function do?

The "len()" function returns a natural number corresponding to the number of characters in a given string.

What does "hello"[3] equal to?

"hello"[3] equals to the fourth character in the string "hello" which is "l". The string index starts at 0 and works its way up increasing by 1 everytime. So had the function been "hello"[0] the result would have been "h".
